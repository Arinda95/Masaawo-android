Searching number plates.

Searching number plates using masaawo is a quite easy, Input search utilizes a text box on the first page of masaawo, input the number plate you would like to know about and click the search button (magnification glass). A response will return with a negative or positive response.

Reporting number plates.

Reporting number plates is unfortunately disabled in the mobile application for security reasons, you send the numberplate and all necessary information to www.masaawo5@gmail.com or you can report a numberplate by going to www.masaawo.com, log in with your personal phone number and secure password (cannot be changed so remember it) Then you can head to 'report a vehicle' and fill in the form. You will be contacted to verify information.
